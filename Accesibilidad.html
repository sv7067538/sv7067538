Ayuda de accesibilidad dentro de nuestro sitio web

Nuestro sitio web de noticias está comprometido con la accesibilidad para todos los usuarios, independientemente de sus capacidades físicas o tecnológicas. Queremos garantizar que todas las personas puedan acceder y disfrutar de nuestros contenidos de manera fácil y eficiente. A continuación, te ofrecemos una guía sobre las herramientas de accesibilidad disponibles en nuestro sitio y algunos consejos para mejorar tu experiencia de navegación.

  1. Navegación con Teclado

    2. Puedes navegar por nuestro sitio utilizando solo el teclado. Usa la tecla Tab para moverte entre los enlaces, botones y campos interactivos.

    Utiliza Shift + Tab para retroceder en la navegación.

    Pulsa Enter para seleccionar un elemento destacado, y Esc para cerrar menús o ventanas emergentes.

  2. Compatibilidad con Lectores de Pantalla

    3. Nuestro sitio está optimizado para trabajar con lectores de pantalla como NVDA, JAWS y VoiceOver. Esto incluye descripciones claras para imágenes, encabezados correctamente estructurados y etiquetas para botones y enlaces.

    Las etiquetas de las imágenes y los textos alternativos ayudan a los usuarios con discapacidad visual a comprender el contenido multimedia.

  3. Modo de Alto Contraste

    4. Para mejorar la legibilidad, puedes activar el Modo de Alto Contraste. Esto es útil para usuarios con baja visión o sensibilidad a los colores. Encuentra esta opción en la esquina superior derecha de nuestra página o en la configuración de accesibilidad de tu navegador.

  4. Tamaño de Fuente Ajustable

    5. Nuestro sitio permite ajustar el tamaño del texto para mejorar la lectura. Usa los controles de zoom de tu navegador (Ctrl + para aumentar, Ctrl - para disminuir) o accede a nuestras configuraciones para establecer un tamaño de fuente predeterminado.

  5. Subtítulos y Transcripciones

    6. Los videos incluidos en nuestras noticias vienen con subtítulos para aquellos con dificultades auditivas. Además, ofrecemos transcripciones completas para contenido multimedia, asegurando que no te pierdas ningún detalle importante.

  6. Atajos de Teclado

    7. Ofrecemos atajos rápidos para una navegación eficiente:

    • S: Ir al contenido principal
    • H: Ir al encabezado de la página
    • L: Lista de artículos
    • 0: Acceder a la página de ayuda
  7. Compatibilidad Móvil

    8. Nuestro sitio es responsivo, lo que significa que se adapta automáticamente a dispositivos móviles y tabletas, garantizando una experiencia fluida y accesible en todos los tamaños de pantalla.

  8. Soporte Técnico

    9. Si encuentras alguna dificultad para acceder al contenido o si necesitas soporte adicional, no dudes en contactarnos a través de nuestro formulario de Atención al Cliente o mediante un correo electrónico. Estamos aquí para ayudarte y mejorar continuamente nuestro sitio en función de tus comentarios.

  9. Consejos Adicionales

    10. Recomendamos usar navegadores actualizados para garantizar la mejor experiencia de accesibilidad.

    Puedes activar opciones de accesibilidad adicionales en la configuración de tu sistema operativo, como la ampliación de pantalla, el uso de colores invertidos, y más.

Nos esforzamos por cumplir con las pautas de accesibilidad web, incluyendo las Pautas de Accesibilidad al Contenido en la Web (WCAG 2.1), para ofrecer una experiencia inclusiva para todos. Agradecemos tus sugerencias y observaciones para seguir mejorando. ¡Gracias por visitar InfoSistemas!
